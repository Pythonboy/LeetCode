# Pascal's Triangle II  
Given a non-negative index k where k â‰¤ 33, return the kth index row of the Pascal's triangle.
Note that the row index starts from 0.

```
Input: 3
Output: [1,3,3,1]
```

**C++**
```
class Solution {
public:
    vector<int> getRow(int rowIndex) {
        vector<int>Res;
        for(int i=0 ; i<=1 ; i++)
            Res.push_back(i);
        Res.push_back(0);
        for(int i=0 ; i<rowIndex ; i++)
        {
            vector<int>Temp;
            Temp.push_back(0);
            int len = Res.size();
            for(int j=1 ; j<len ; j++)
            {
                Temp.push_back(Res[j]+Res[j-1]);
            }
            Temp.push_back(0);
            Res = Temp;
        }
        Res.pop_back();
        auto j = Res.begin();
        Res.erase(j);
        return Res;
    }
};
```
